# 기능 완료 체크리스트

## 요구사항

- [ ] 대상 FR/NFR/AC 식별자가 명확하다.
- [ ] 포함 범위와 제외 범위를 기록했다.
- [ ] 사용자가 관찰 가능한 완료 조건이 있다.

## 구현

- [ ] Unity 콘솔에 새 오류나 경고를 만들지 않았다.
- [ ] 씬·프리팹·직렬화 참조가 유지된다.
- [ ] 프레임마다 불필요한 검색·할당을 만들지 않았다.
- [ ] 사망·비활성·일시정지 같은 경계 상태를 처리했다.
- [ ] 구현을 담당자가 설명하고 수정할 수 있다.

## 검증

- [ ] 최소 테스트 씬에서 통과했다.
- [ ] 통합 씬에서 통과했다.
- [ ] 관련 자동 테스트가 통과했다.
- [ ] 회귀 시나리오를 확인했다.
- [ ] 기대 결과와 실제 결과가 문서에 남아 있다.

## 교육 기록

- [ ] 원문 프롬프트와 버전을 기록했다.
- [ ] AI 결과의 채택·수정·기각 근거를 기록했다.
- [ ] 개발일지에 학습 포인트를 적었다.
- [ ] 오류가 있었다면 재현과 해결 방법을 기록했다.
- [ ] 코드·프롬프트·테스트·문서 링크가 연결되어 있다.

## 배포 가능성

- [ ] 외부 에셋의 라이선스를 확인했다.
- [ ] 대상 플랫폼에서 플레이할 수 있다.
- [ ] 기능을 끈 상태 또는 초기 상태도 정상이다.
- [ ] 완료 커밋을 식별할 수 있다.
